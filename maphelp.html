How to Use the Regional In-Store Pick Up Availability Map

The Regional In-Store Pick Up Availability Map (Canada) lets you quickly check if an item is available for local pick up in various provinces and territories. Simply hover or click on a region in the map image, and an availability message will appear. If pick up is offered in that area, the region will show as Available. If not, it’ll show as Unavailable.

Each region’s availability is updated dynamically on the page, ensuring you have the most current information. For example, you may see messages such as:

  • Ontario: In-store pick up available in Ottawa.
  • Nunavut: In-store pick up not available.

These messages help you determine whether to visit a particular location or explore shipping options instead.

You can always return to the main map page for a larger view or to select a different region. If you have questions or need further assistance, please contact our customer support.
